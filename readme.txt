== Description ==

Any news ticker to show anything as ticker on your website very easily. you can use it in any page, post or in any widget of your website by simple short-code. you can also customise the ticker in shortcode. No need technical or coding knowledge.

[**View samples of each mode.**](https://demo.mucasoft.com/wp/plugins-demo/any-news-ticker-demo/)



== Installation ==

1. Upload `anyNewsTicker` directory to the "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" menu in WordPress
3. Use [any-ticker] in any page, post or widget.

That's all...

== Default settings ==

1. Display latest 5 posts from any category.
2. Ticker speed .10 (ms)
3. Left-right control "On"
4. Default ticker title "Latest Posts"


== Customisation ==

You can customise your ticker if you want, read bellow carefully.
1. Use category_id="2" to show specific category post.

For example: [any-ticker category_id="2"]
